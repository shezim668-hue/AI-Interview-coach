# 🤖 AI Interview Coach

AI Interview Coach is an AI-powered web application built with Python and Streamlit that helps job seekers prepare for interviews more effectively. The application uses Google Gemini AI to analyze resumes, generate personalized interview questions, and create professional cover letters based on a job description.

## ✨ Features

* 📄 AI Resume Analysis
* 📊 ATS-style Resume Evaluation
* 🎯 Job Description Matching
* 🎤 AI-Generated HR & Technical Interview Questions
* 📝 AI Cover Letter Generator
* 🤖 Powered by Google Gemini AI

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Gemini AI API
* PyPDF
* python-dotenv

## 🚀 How It Works

1. Upload your resume in PDF format.
2. Paste the job description.
3. Get an AI-powered resume analysis.
4. Generate personalized HR and technical interview questions.
5. Create a professional cover letter tailored to the job.
